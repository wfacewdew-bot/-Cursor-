# 英语六级单词弹幕软件

这是一个在 Windows 桌面上显示 **英语六级单词弹幕** 的小工具：

- 使用 **本地 SQLite 数据库** 存储单词；
- 第一次运行时自动从网络下载 **英语六级词库（CET-6）** 并写入本地数据库；
- 在屏幕顶部 **全屏透明窗口** 中不断滚动显示单词；
- 可以在 `config.py` 中自行设置 **弹幕速度** 和 **同时在屏幕上的单词数量** 等参数。

## 环境准备

1. 安装 Python（推荐 3.10+）。
2. 在项目目录（`Bullet Screen`）中运行：

```bash
pip install -r requirements.txt
```

## 运行方式

在项目目录下执行：

```bash
python main.py
```

第一次运行：

- 程序会自动下载六级词库文本；
- 解析其中的英语单词；
- 初始化本地数据库 `words.db`；
- 然后启动全屏弹幕窗口。

之后再次运行会直接使用本地数据库，不会重复下载。

## 参数配置

打开 `config.py` 可以修改：

- 弹幕滚动速度；
- 同屏显示的单词数量；
- 单词刷新间隔等。

修改保存后重新运行 `main.py` 即可生效。
